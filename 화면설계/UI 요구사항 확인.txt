UI - 사용자 인터페이스
UX - 사용자 경험

UI 유형
1. CLI - 정적 텍스트 기반 (명령어를 입력)
2. GUI - 그래픽 환경 기반 (마우스, 전자펜)
3. NUI - 직관적 사용자 기반 (신체부위 이용)
4. OUI - 유기적 상호작용 기반 (사물을 입출력장치로)

UI 설계 원칙
1. 직관성 - 누구나 쉽게 이해하고 사용할 수 있어야 함
2. 유효성 - 정확하게 사용자의 목표 달성될 수 있게
3. 학습성 - 초보와 숙련자 모두 쉽게 배우고 사용할 수 있게
4. 유연성 - 사용자의 요구사항을 최대한 수용

UI 지침
1. 3C 분석 - Customer, Company, Competitor 을 비교하고 분석
2. SWOT 분석 - Strength, Weakness, Opportunity, Threat 요인을 규정
3. 시나리오 플래닝 - 불확실성 높은 상황 변화를 사전에 예측
4. 워크숍 - 소집단의 인원이 새로운 지식이나 방법을 서로 교환

사용성 테스트 - 사용자가 직접 제품 사용, 시나리오에 맞춰 과제를 수행

UI 화면 설계
1. 와이어프레임 - 이해관계자들과의 화면 구성 협의, 화면 단위의 레이앋웃 설계
2. 스토리보드 - 서비스 구축을 위한 모든 정보가 담겨있는 설계 산출물
3. 프로토타입 - 정적인 화면으로 설계된 와이어프레임, 스토리보드에 동적 효과 적용
